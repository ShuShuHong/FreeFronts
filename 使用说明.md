# FreeFronts 使用说明

## 简介

FreeFronts 是一款字体预览与下载工具，收录 **78 款可免费商用中文字体**（2026.4 版本），按风格分为 7 大类：黑体、宋体/仿宋、楷体、标题/创意、手写体、书法/艺术、圆体。

所有字体文件均自托管，无外部 CDN 依赖，浏览器打开即可使用。

---

## 基本操作

### 选择字体
- 左侧列表中点击任意字体即可选中预览
- 列表顶部的搜索框支持按**字体名称**或**序号**快速筛选
- 鼠标滚轮滚动浏览完整列表

### 预览字体
- 上方文本框输入或粘贴任意文字，预览区实时渲染
- 点击预设按钮快速切换示例文本：**千字文 / 唐诗 / 中英混排 / 数字符号 / 设计文案**
- 左侧滑块调节**字号**（12–200px）和**行高**（1.0–3.0）

### 下载字体
- 选中字体后，底部「下载字体」按钮下载当前字体的 `.ttf` / `.otf` 文件
- 下载文件命名格式：`{序号}_{字体名}.扩展名`（如 `01_思源黑体.ttf`）
- 右上角「全部字体打包下载」可一键下载全部 78 款字体的 ZIP 包（约 290MB）

### 复制字体名
- 点击「选择字体」旁的 **复制字体名称** 按钮，将当前字体的序号+名称（如 `01 思源黑体`）复制到剪贴板

## 部署说明

1. 将整个 `FreeFronts` 文件夹放到 Web 服务器目录下
2. 确保 `fonts/` 子目录中的所有字体文件可被服务器读取（权限 644）
3. 访问 `index.html` 即可使用

无需 Node.js、PHP 或数据库——纯静态 HTML 页面。

---

## 添加新字体

在 `index.html` 的 `<script>` 中找到 `var fonts=[...]` 数组，按以下格式添加：

```js
{id:'f80',num:'80',name:'新字体名',cat:'hei',file:'fonts/80_新字体.ttf',ext:'truetype'},
```

`cat` 可选值：`hei`（黑体）、`song`（宋体）、`kai`（楷体）、`title`（标题）、`hand`（手写）、`art`（书法）、`round`（圆体）。

同时在 `<style>` 中添加对应的 `@font-face` 声明：

```css
@font-face {
  font-family: '新字体名';
  src: url('fonts/80_新字体.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```
